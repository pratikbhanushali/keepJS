KeepJS

Keep is a simple Javascript Library used to manage your Browser Storage. It uses Cookies, Local Storage and Session Storage for this purpose

Welcome to GitHub Page for KeepJS.

KeepJS is built with a simple use case in mind of efficiently maintaining and using web browser storage. Keep works well with Cookies, Local Storage and Session Storage.

Functionalities:

  1. SET Data:

Method name: keep.set()

Parameters: name Identifier used to store a value in browser storage. value Value to be stored in the browser storage. storage Optional: Defaults to Cookies. Possible values are 'C'(Cookies) / 'L'(localStorage) / 'S'(session storage). sess Optional: Defaults to true. Used to set cookie as session cookie or persistant cookie. Possible values are true / false. expires Optional: Sets a cookie with an expiry. This key is used only when @sess = true. Use the Javascript Date function to pass expiry E.g. ("Mon, 4 Nov 2100 13:00:00 GMT") path Optional: Defaults to '/'. encrypt Optional: This key if set to boolean true, will encrypt data that goes into store.

P.S : encrypt - DO NOT SET THIS TO true IF YOU WANT TO USE COOKIE DATA SERVER-SIDE.

Example :

keep.set({ name: "cookie-name", value: "Hello World!", sess: false, expires: "Mon, 4 Nov 2100 13:00:00 GMT", encrypt: true });
